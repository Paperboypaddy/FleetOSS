# FleetOSS

> Open Source Fleet & Asset Intelligence Platform

FleetOSS is a modern, self-hostable platform for tracking vehicles, trailers, equipment, and other GPS-enabled assets.

Built with a focus on openness, extensibility, and ownership, FleetOSS provides real-time tracking, trip history, telemetry, maintenance records, and asset management without vendor lock-in.

Whether you're an independent contractor with a single van, a small business managing a fleet, or a developer building custom telematics solutions, FleetOSS gives you complete control over your data and infrastructure.

---

## Why FleetOSS?

Most fleet management platforms are:

- Expensive
- Closed-source
- Contract-driven
- Designed for large fleets
- Difficult to customize
- Locked into proprietary hardware

FleetOSS takes a different approach.

### FleetOSS is:

✅ Open Source

✅ Self-hostable

✅ API-first

✅ Hardware agnostic

✅ Privacy focused

✅ Built for fleets of any size

✅ Designed for developers and operators alike

## Hardware Philosophy

FleetOSS is designed to support multiple hardware vendors and protocols.

The goal is simple:

> If a device can provide location or telemetry data, FleetOSS should be able to ingest it.

Planned support includes:

- Mobile devices
- OBD-II adapters
- Cellular GPS trackers
- Asset trackers
- Custom integrations
- Community-developed device adapters

---

## Architecture

FleetOSS is being designed around a simple principle:

> Raw telemetry in, actionable information out.

Core components include:

- API Layer
- Asset Management
- Telemetry Ingestion
- Trip Processing
- Geospatial Storage
- Real-Time Event Streaming
- Web Dashboard
- Mobile Applications

## Vision

FleetOSS began as a personal project to solve a simple problem:

> I wanted a modern, open-source platform to track my vehicles, equipment, mileage, maintenance history, and telemetry without relying on expensive proprietary software.

Over time, the vision expanded.

FleetOSS aims to become a complete open-source platform for fleet operations, asset intelligence, and vehicle telemetry that anyone can deploy, modify, and own.
